# CNRoutingAssignment

## Module: Computer Networks
This repository contains the practical implementations for the Router Configuration and Routing Protocols assignment using Cisco Packet Tracer.

### Files Included:
1. Task2_Static_Routing.pkt - Implementation of a static routing configuration across a 3-router triangular topology (Computing, Business, and Science networks).
2. Task3_PartA_RIP.pkt- Dynamic routing implementation using Routing Information Protocol (RIPv2) on a linear 3-router topology (Kandy, Colombo, and Galle).
3. Task3_PartB_EIGRP.pkt - Dynamic routing implementation using Enhanced Interior Gateway Routing Protocol (EIGRP) on the same linear topology.

### Verification
All configurations have been verified using end-to-end ICMP ping tests to ensure complete network connectivity across all local and wide-area networks.
